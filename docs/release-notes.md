## 1.0.0 (2024-08-19)

- Initial public release of the Replicate Obsidian plugin.
- Generate images from a selection or from a prompt entered in a modal.
- Configurable Replicate model (`<owner>/<name>` or `<owner>/<name>:<version>`).
- Free-form JSON input passed to the chosen model.
- Optional: copy the generated output to the clipboard.
- Optional: append the generated output to the current note.
